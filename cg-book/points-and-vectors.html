Points and vectors

We will assume two-dimensional points and vectors $(x, y)$ and $\begin{bmatrix}x\\ y\end{bmatrix}$ respectively.

However, everything we are going to state is true for points in higher dimensions as well.

Points will be denoted $(x, y)$ whereas vectors will be denoted $\begin{bmatrix}x\\y\end{bmatrix}$.

Vector between two points

When we have two points $P_1 = (10, 10)$ and $P_2 = (40, 50)$ we can calculate a vector $\vec{v}$ between these points by subtracting them.

$$\begin{align} \vec{v} &= P_2 - P_1\\ &= (40 - 10, 50 - 10)\\ &= \begin{bmatrix}30\\40\end{bmatrix} \end{align}$$

Translating a point

When we add a vector to a point we translate to a new point. If we have a point $P_1$ and a vector $\vec{v}$ we can calculate $P_2$ with

$$P_2 = P_1 + \vec{v}$$

Suppose we have a point $P_1 = (10, 10)$ and we translate it by the translation vector $\vec{v} = {\begin{bmatrix}20\\30\end{bmatrix}}$.

We would end up with $P_2$ being

$$\begin{align} P_2 &= P_1 + \vec{v}\\ &= (10, 10) + \begin{bmatrix}20\\30\end{bmatrix}\\ &= (10 + 20, 10 + 30)\\ &= (30, 40) \end{align}$$
